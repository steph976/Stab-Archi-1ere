Dans ce type de document les informations sont inscrites sous forme de tableau.

Lecture du document

Pour un tirant d'eau donné (dans l'exemple ci dessous Tm = 2,05 m), le tableau indique que le déplacement qui correspond est D=102,79 tonnes

Inversement pour un déplacement D = 102,79 tonnes, le tableau indique un tirant d'eau moyen Tm = 2,05 m

Il en va de même pour les autres informations, par exemple :

Si le déplacement du navire est de 107,14 tonnes alors KB la hauteur du centre de carène est de 1,313 m

Interpolation

Pour des valeurs entre deux lignes de la table, les résultats ne sont pas directement lisible. Il faut interpoler. par exemple ici, quel est le tirant d'eau correspondant à un déplacement de 141 tonnes ?
